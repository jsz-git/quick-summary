# 快捷总结笔记 - 前端功能需求总结

## 🎯 核心功能模块

### 1. ⚙️ 配置管理

#### 1.1 快捷键设置
- **功能**: 可视化修改快捷键
- **选项**:
  - Cmd/Ctrl + Shift + S（默认）
  - Cmd/Ctrl + Alt + S
  - 自定义组合键
- **验证**: 检测快捷键冲突
- **实时生效**: 修改后立即生效，无需重启

#### 1.2 运行模式选择
- **全自动模式**（自动选中 + 总结）
  - 自动全选当前区域
  - 自动复制到剪贴板
  - 自动调用AI总结
  - 适用场景：快速总结

- **半自动模式**（单次运行）
  - 手动复制内容
  - 点击按钮或按快捷键触发
  - 适用场景：精确控制

- **定时模式**（可选）
  - 定时总结剪贴板内容
  - 适合长期监控场景

#### 1.3 AI模型配置
- **模型选择器**:
  - DeepSeek（推荐，性价比高）
  - 通义千问（中文优化）
  - Claude（高质量）
  - OpenAI GPT（生态完善）
  - 本地模型 Ollama（免费）

- **模型参数**:
  - max_tokens（输出长度）
  - temperature（创造性）
  - 自定义API端点

- **API密钥管理**:
  - 密钥输入框（加密存储）
  - 连接测试按钮
  - 余额查询（可选）

### 2. 📝 笔记管理系统

#### 2.1 笔记列表
- **展示方式**:
  - 列表视图（文件名、时间、预览）
  - 卡片视图（缩略图、摘要）
  - 时间线视图（按日期分组）

- **排序选项**:
  - 按时间（最新/最旧）
  - 按大小
  - 按AI模型

- **筛选功能**:
  - 按日期范围
  - 按AI模型
  - 按关键词

#### 2.2 笔记详情
- **Markdown渲染**:
  - 实时预览
  - 代码高亮
  - 表格支持

- **操作功能**:
  - 编辑（修改内容）
  - 导出（PDF/HTML/TXT）
  - 分享（复制链接）
  - 删除（带确认）

#### 2.3 笔记搜索
- **全文搜索**:
  - 标题搜索
  - 内容搜索
  - 正则表达式支持

- **高级搜索**:
  - 按日期范围
  - 按AI模型
  - 按标签（可选）

- **搜索结果**:
  - 高亮显示匹配内容
  - 快速定位

### 3. 🤖 总结功能

#### 3.1 手动触发
- **一键总结按钮**:
  - 读取剪贴板
  - 显示预览
  - 确认后生成

- **自定义提示词**（高级）:
  - 修改总结模板
  - 保存常用模板
  - 导入/导出模板

#### 3.2 批量处理
- **批量总结**:
  - 选择多个对话文件
  - 批量生成总结
  - 进度显示

#### 3.3 总结历史
- **最近总结**:
  - 显示最近10条
  - 快速访问

### 4. 📊 统计与分析

#### 4.1 使用统计
- 总总结次数
- 各模型使用频率
- 平均总结长度
- 节省时间估算

#### 4.2 成本统计
- API调用次数
- Token使用量
- 费用估算（按模型）

### 5. 🔧 系统设置

#### 5.1 通用设置
- **输出目录**: 自定义保存位置
- **文件命名**: 时间戳/自定义格式
- **通知**: 开启/关闭系统通知
- **声音**: 提示音开关

#### 5.2 外观设置
- **主题**: 亮色/暗色/跟随系统
- **字体大小**: 小/中/大
- **语言**: 中文/英文

#### 5.3 高级设置
- **自动启动**: 开机自启动
- **后台运行**: 最小化到托盘
- **日志级别**: DEBUG/INFO/WARNING/ERROR

### 6. 🔐 安全与隐私

#### 6.1 数据管理
- **数据导出**: 导出所有笔记和配置
- **数据清理**: 清理旧笔记（按时间）
- **数据备份**: 自动备份（可选）

#### 6.2 隐私设置
- **脱敏处理**: 移除敏感信息
- **本地模式**: 优先使用本地模型

## 🎨 UI/UX 设计要求

### 界面布局
```
┌─────────────────────────────────────────┐
│  🎯 快捷总结笔记          [─] [□] [×]   │
├──────────┬──────────────────────────────┤
│          │                              │
│  📝 笔记  │      主内容区                │
│  ⚙️ 配置  │   (笔记列表/编辑/设置)       │
│  🤖 总结  │                              │
│  📊 统计  │                              │
│  🔧 设置  │                              │
│          │                              │
│          │                              │
├──────────┴──────────────────────────────┤
│  状态栏: 就绪 | 模型: DeepSeek | v1.1.0 │
└─────────────────────────────────────────┘
```

### 交互设计
- **一键操作**: 核心功能不超过3次点击
- **快捷键支持**: 常用操作有快捷键
- **拖拽支持**: 拖拽文件到窗口自动总结
- **右键菜单**: 快速操作

### 视觉风格
- **简洁现代**: Material Design 或 Fluent Design
- **响应式**: 适配不同屏幕尺寸
- **动效**: 适度动画，不过度

## 🔧 技术要求

### 性能
- 启动时间 < 2秒
- 内存占用 < 100MB（前端部分）
- 响应时间 < 100ms

### 兼容性
- macOS 10.14+
- Windows 10+
- Linux（主流发行版）

### 安全
- API密钥加密存储
- 本地数据加密（可选）
- 不联网功能可离线使用
