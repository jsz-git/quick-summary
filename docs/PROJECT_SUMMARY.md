## 🎯 解决的问题

### 原始需求
> "有没有一个总结工具，当我在对话中说'可以了'它能帮我主动记录本次完成的步骤、日志、及其中使用的命令行"

### 改进方案
1. ✅ **安全性** - 不监听对话，用户主动按快捷键触发
2. ✅ **便捷性** - 自动选中内容，无需手动复制
3. ✅ **轻量化** - 内存 <10MB，CPU <1%
4. ✅ **多模型** - 支持5+ AI模型，可选择最经济的

## 🚀 快速开始

### 3步启动

```bash
# 1. 安装
cd ~/Desktop/快捷总结笔记
pip3 install -r requirements.txt

# 2. 配置
cp .env.example .env
# 编辑 .env，填入 DEEPSEEK_API_KEY

# 3. 启动
python3 src/main.py
```

### 使用方法

```
1. 在任意AI对话工具中（元宝、千问、Claude Code）
2. 将光标放在对话区域
3. 按 Cmd+Shift+S（Mac）或 Ctrl+Shift+S（Windows）
4. 完成！总结已保存到 logs/ 目录
```
