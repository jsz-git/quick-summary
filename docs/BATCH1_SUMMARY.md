## 🔧 技术实现

### 工作流程
```
用户复制对话 → 按快捷键 → 读取剪贴板 → 调用AI → 生成总结 → 保存文件 → 发送通知
```

### 依赖库
- `anthropic` - AI接口
- `keyboard` - 快捷键监听
- `pyperclip` - 剪贴板操作
- `pyyaml` - 配置管理
- `python-dotenv` - 环境变量

## 📝 下一步操作

1. **安装依赖**
   ```bash
   cd ~/Desktop/快捷总结笔记
   ./install.sh
   ```

2. **配置API密钥**
