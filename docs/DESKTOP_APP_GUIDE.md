## 🔧 故障排除

### 问题1：无法访问http://localhost:8080

**解决**：
```bash
# 检查服务是否运行
ps aux | grep python

# 重新启动
./start-full-web.sh
```

### 问题2：笔记加载失败

**解决**：
```bash
# 确保API服务运行
curl http://localhost:5000/api/notes

# 如果失败，检查日志
python3 api_server.py
```

### 问题3：总结功能不工作

**解决**：
1. 检查API密钥配置（.env文件）
2. 确保Python后端正常
3. 查看浏览器控制台错误
