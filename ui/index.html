快捷总结笔记

AI 智能对话总结工具

🚀 首次使用指南

1️⃣
配置 AI 模型

点击侧边栏「配置」，选择你的 AI 提供商（DeepSeek/OpenAI/Claude/Qwen）并填写 API Key。

2️⃣
笔记存储位置

在「配置」页面可以修改笔记保存目录，默认保存在项目目录的 logs 文件夹中。

3️⃣
设置快捷键

在「配置」页面可以修改全局快捷键，按下快捷键可快速触发总结功能。

💡 两种工作模式

⚡ 快捷键模式

在任意应用中选中内容，按下 Ctrl+Shift+S（或自定义快捷键），自动全选并复制内容，生成总结。

📋 手动模式

复制需要总结的内容到剪贴板，点击「一键总结」按钮生成总结。

📖 快速开始

1. 先在「配置」中完成 API Key 配置

2. 复制任意 AI 对话内容

3. 点击「一键总结」或使用快捷键

4. 总结内容将自动保存到笔记中

🤖 AI总结

一键生成对话总结

⚙️ 选择工作方式

📋 使用步骤：

  1. 在其他对话页面用鼠标选中需要总结的内容
  2. 按 Cmd+C 复制到剪贴板
  3. 回到此页面，点击下方"一键总结"按钮
  4. 等待AI生成总结（约10-30秒）

💡 提示：确保已复制对话内容到剪贴板

⌨️ 快捷键总结

在任何对话页面按快捷键：

Cmd + Shift + 1

系统会自动选中当前页面内容并生成总结

💡 可在配置页面修改快捷键
